# Bookmark Management Strategy

Guidelines for managing browser bookmarks across machines using zero-lock-in principles.

## 1. Core Principles
- **No Proprietary Tools:** Avoid browser-specific sync if possible, or use it only as a transient layer.
- **Markdown-First:** Maintain a master list of essential, project-specific, or category-based bookmarks in this repository.
- **Category Based:** Organise by function (e.g., *Reference*, *Assets*, *Clients*, *Admin*).
