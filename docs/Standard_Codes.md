# Standard Material & Asset Codes

This document defines the standardized abbreviations used for naming materials, colors, furniture, and lighting across all 3D assets and project files. 

Using these codes ensures consistency, making it easier to parse scenes programmatically and maintain a clean library.

## 1. Primary Material Categories
Use these as the primary prefix when naming materials (e.g., `MI_WD_Oak_Light`).

| Code | Material Type | Notes |
| :--- | :--- | :--- |
| **WD** | Wood | Includes solid wood, veneers, and wood flooring. |
| **MT** | Metal | General metal (use specific sub-codes below if needed). |
| **AL** | Aluminium | |
| **SS** | Stainless Steel | |
| **GL** | Glass | |
| **LA** | Laminate | Plastic laminates, Formica, etc. |
| **FB** | Fabric | Upholstery, curtains, linens. |
| **ST** | Stone | Marble, granite, reconstituted stone. |
| **PT** | Paint | Wall paint, lacquers. |
| **TL** | Tiles | General tiles. |
| **WT** | Wall Tile | Specific to vertical surfaces. |
| **FT** | Floor Tile | Specific to horizontal surfaces. |
| **SHP** | Parallax Shopfront | OSL/Parallax mapping textures. |
| **ER** | Earth | Soil, dirt, landscaping ground cover. |
| **LT** | Light / Emissive | Emissive materials for bulbs, screens. |
